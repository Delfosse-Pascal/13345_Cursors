Bienvenue. Vous explorez une collection de 13 345 curseurs Windows classés automatiquement en 54 tiroirs de 250 fichiers chacun. Chaque tiroir renferme une sélection variée de pointeurs animés (.ani) et statiques (.cur) issus de sources publiques diverses.

Cliquez sur un tiroir ci-dessous pour découvrir ses curseurs, les prévisualiser, et les enregistrer sur votre machine d'un clic droit → Enregistrer sous.

Bonus — Script de rangement

Un petit utilitaire Batch Windows est inclus : il range automatiquement n'importe quel ensemble de fichiers en tiroirs numérotés de 250 fichiers chacun. C'est ce script qui a servi à organiser cette collection.

⤓ Télécharger le script .bat

Comment se servir & installer un curseur

  1. Choisissez un dossier (un « tiroir ») dans la grille ci-dessous, parcourez la galerie et survolez les cartes : le pointeur de votre souris prend l'apparence du curseur sous le pointeur.
  2. Faites Clic droit sur le bouton Télécharger d'un curseur, puis choisissez Enregistrer la cible du lien sous… (ou Enregistrer sous…) pour le conserver sur votre PC.
  3. Sous Windows : Clic droit sur le fichier .cur ou .ani téléchargé, puis Installer dans le menu contextuel.
  4. Ou alors : Panneau de configuration → Souris → onglet Pointeurs, sélectionnez un rôle (Sélection normale, Aide, etc.), cliquez Parcourir… et choisissez le curseur. Enregistrer sous… pour créer votre schéma.
